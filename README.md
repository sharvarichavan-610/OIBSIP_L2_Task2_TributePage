# Tribute Page

## 📌 Project Overview
This project is a Tribute Page created as Task 2 of Level 2 during the internship at Oasis Infobyte.
The webpage is designed to honor and present information about an inspiring personality using HTML and CSS.

## 🚀 Features
Attractive webpage layout
Image and information about the personality
Clean and structured content
Responsive design

## 🛠️ Technologies Used
HTML – Structure of the webpage
CSS – Styling and layout design

## Learning Outcome:
Successfully created a tribute webpage and improved skills in HTML structure and CSS design.
